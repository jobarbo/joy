== Description ==
Azalea is a clean and elegant WordPress blog theme, perfect for lifestyle, food or fashion blogging. 100% responsive, customizable and easy to use.

* Responsive Layout
* Social links in the site header and footer
* Optional author information box in the single post
* Right or left optional sidebar
* Authors page template
* No sidebar page template
* Full-width page template
* Gallery and video post formats
* Custom header
* Custom Colors
* Jetpack infinite scroll support
* Contact Form 7 support
* WP Instagram Widget support

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Azalea in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
